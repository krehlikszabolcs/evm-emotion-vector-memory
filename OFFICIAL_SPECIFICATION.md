# Official Specification (Canonical)

The authoritative normative specification of Emotion Vector Memory (EVM)
is maintained via DOI releases on Zenodo.

Current official release:
EVM v2.1 — Unified Directed Vector Identity Standard
https://zenodo.org/records/18664771

GitHub repositories provide reference materials, examples, and templates.
The Zenodo DOI release remains the canonical source of the standard.
