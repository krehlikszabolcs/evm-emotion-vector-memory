# Contributing

Thanks for your interest in EVM.

Important:
- The EVM core ontology is closed and fixed (v2.0+).
- Contributions are welcome for:
  - documentation clarity
  - reference examples
  - integration guides
  - tooling (validators, log readers)
- Proposed changes that alter the core ontology or introduce new structural dimensions
  will not be accepted.

By submitting a contribution, you agree that it may be incorporated into the repository
under the repository’s license and notices, without granting any commercial rights.

For partnership / pilot requests, use the Partnership template in Issues.
